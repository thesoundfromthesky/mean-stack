Intro about this website

Overview

1.Backend is written in Typescript with NodeJs, Express, and Mongoose. It's a RESTful API that supports create, update, read, and delete of user accounts and forum posts. SocketIO is used for Chat server.

2.Angular material is used for frontend.

User

1.User accounts can be created, read, updated, and deleted through User menu.

Forum

1.Users can create, read, update, and delete posts through Forum menu.

Chat

1.Users can send messages to other users in chat.
